    binaire
    Version binaire
        Pour faciliter l'utilisation de Mongrey, des binaires autonomes sont livrés pour chaque 
        déclinaisons de Mongrey
        
        Chaque binaire est préparer à l'aide de :pypi:`PyInstaller` et n'a besoin d'aucune installation 
        supplémentaire, en dehors du backend (MongoDB, PostgreSQL ou MySQL).
